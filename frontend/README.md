# frontend

We are rocking with vue, bun, and bulma.

I created the project with bun, selecting TypeScript

```sh
bun create vue@latest
```

## Project Setup

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun dev
```

### Type-Check, Compile and Minify for Production

```sh
bun run build
```
